FTD Quantitative Research

Python 3.10+ · zero runtime dependencies · 22-test suite · MIT

A deterministic, strategy-agnostic backtesting engine built to answer one question rigorously: does a popular retail trading method contain a real edge? The platform reached a documented negative result — and honored it. The engineering discipline used to get there honestly is the asset on display.

  • Byte-for-byte reproducible backtest runs, hash-stable under SHA-256.
  • Anti-curve-fit methodology — pre-registration, out-of-sample validation, transaction-cost modeling, random-entry research harness.
  • Clean strategy interface keeping trading logic behind a seam; test suite runs on the standard library alone.
